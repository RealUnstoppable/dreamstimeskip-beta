# Bolt Learnings

- **Firebase Cloud Functions & Promise.all**: Always use `Promise.all()` to handle multiple async operations (like API requests or database updates) to execute them concurrently instead of using sequential loops (like `for...of`). This minimizes execution time by executing tasks in parallel, drastically speeding up I/O-bound operations and avoiding function timeouts. It can make a significant difference, for example taking 50 parallel queries down to 50ms from 2.5 seconds sequentially.
